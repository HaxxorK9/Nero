# Nero

**Nero** is an advanced Discord bot designed for roleplay communities.

It provides character management, user statistics, interactive commands and persistent database storage.

![NeroBot](https://github.com/HaxxorK9/Nero/blob/main/Nero%20bird.jpg?raw=true)

## Features

### Character System

- Character creation
- Character statistics
- Character progression
- Data persistence

### User Management

- User profiles
- Statistics tracking
- Progress monitoring

### Roleplay Features

- Interactive commands
- Character interactions
- Story progression tools

### Database

- Persistent storage
- Automatic data saving
- User and character management

## Technologies

- SQL Database
